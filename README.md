## Descripción del sistema

Este proyecto consiste en un sistema de gestión de estacionamiento desarrollado en Python para consola.

El sistema permite:

* Registrar el ingreso de vehículos.
* Registrar el egreso de vehículos.
* Controlar la capacidad disponible del estacionamiento.
* Calcular el importe a pagar según las horas de permanencia.
* Mostrar vehículos estacionados.
* Generar estadísticas de uso.
